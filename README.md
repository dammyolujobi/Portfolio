# Olujobi Damilola - React Portfolio

backend developer portfolio. clean, simple, dark mode only.

## 🚀 quick start

```bash
npm install
npm run dev
```

## 🛠️ stack

- React 18
- Vite
- Framer Motion
- vanilla CSS

## ✨ features

- **backend-focused** - showcases APIs, CLIs, and backend work
- **rust project** - todo-cli built in rust
- **code aesthetic** - terminal vibes, monospace fonts
- **dark theme** - cyan/purple accent colors
- **smooth animations** - framer motion
- **simple & fast** - no bloat

## 🎨 design

- dark cyberpunk theme
- code-style comments (// about, // projects)
- casual tone, no corporate speak
- custom cursor glow effect
- responsive design
